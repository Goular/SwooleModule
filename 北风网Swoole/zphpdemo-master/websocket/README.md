websocket Demo运行：
======
    1) git clone https://github.com/shenzhe/zphp

    1) git clone https://github.com/shenzhe/zphpdemo

	3) cd zphp目录

	4) php bin/swoole_http_server.php {zphpdemo完整目录}/websocket

	5) 浏览器运行：http://host:9502/
